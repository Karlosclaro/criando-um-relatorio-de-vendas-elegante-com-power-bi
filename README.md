

## 📊 Criando um Relatório de Vendas Elegante com Power BI

Este projeto tem como objetivo demonstrar a criação de um **relatório de vendas moderno, interativo e visualmente elegante utilizando o Power BI**. Ao longo do desenvolvimento, são exploradas boas práticas de **modelagem de dados, criação de dashboards, uso de KPIs, filtros interativos e visualizações estratégicas** para apoiar a tomada de decisão.

### 🎯 Objetivos do Projeto

*   Transformar dados brutos em informações relevantes
*   Criar visualizações intuitivas e atrativas
*   Analisar indicadores de desempenho de vendas
*   Aplicar conceitos de design e storytelling com dados

### 🛠️ Tecnologias Utilizadas

*   Power BI Desktop
*   DAX básico e intermediário
*   Modelagem de dados
*   Uso de botões interativos
*   Uso de figuras, para aparência dos botões.
